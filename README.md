## 👩‍💻 About Me

I'm an **MS Computer Science student at UW–Madison** with a deep focus on **Machine Learning, Generative AI, and NLP**. I love building systems that push the frontier — from training language models with RLHF to generating ad creatives with diffusion models.

- 🧠 Implementing **RLHF from scratch** — reward modeling, PPO, and LLM alignment
- 🎨 Using **MultiDiffusion** to generate high-quality, context-aware ad images
- 🤝 Built **SquadSync** — a Python-based team collaboration tool
- ✅ Designed an **ADHD Task Manager** in TypeScript with cognitive-accessibility-first UX
- 🔬 Currently diving deep into **foundational models**, fine-tuning, and agent systems
- 🎯 Actively seeking roles in **ML Engineering**, **AI Research**, and **Data Science**
